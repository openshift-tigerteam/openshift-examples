# openshift-examples

Repository for example code and configuration related to OpenShift

## Getting Started

Clone the repo.   
Setup your local venv environment.   
```shell
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```

Start mkdocs.   
```shell
source .venv/bin/activate
mkdocs serve    
```

## Project Creation
```shell
python3 -m venv .venv
source .venv/bin/activate
mkdocs new .
```
